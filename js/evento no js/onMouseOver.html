Evento JS: OnMouseOver e OnMouseOut



OnMouseOver é um evento em JavaScript que é acionado quando o ponteiro do mouse é movido para cima de um elemento específico em uma página da web. O evento pode ser usado para executar uma função JavaScript ou alterar o comportamento de um elemento quando o mouse está sobre ele.
