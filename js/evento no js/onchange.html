Evento JS: onChange



O evento onChange em JavaScript é acionado quando o valor de um elemento HTML, como um campo de formulário, é alterado pelo usuário.

Nesse exemplo, o código JavaScript console.log(this.value) será executado toda vez que o valor do campo de texto for alterado, exibindo o valor atual do campo no console.

PS: lembre-se de clicar no botão ENTER para que o valor do input ir ao console.


Agora se você clicar o botão direito do mouse e clicar em (Inpencionar Elementos) e depois ir em Console, você vera que o resultado do input esta ai.
