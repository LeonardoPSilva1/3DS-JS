Evento JS: OnKeyDown



O evento keydown é uma função do JavaScript que é acionada quando o usuário pressiona uma tecla no teclado. É semelhante ao evento onkeyup, mas é acionado quando a tecla é pressionada, e não quando ela é solta. Você pode usar o evento keydown com um elemento HTML de entrada, como um campo de texto, para executar uma ação enquanto o usuário está digitando. Por exemplo, você pode limitar o número de caracteres que um usuário pode digitar em um campo de entrada, ou verificar se a tecla pressionada é uma tecla específica.

Por exemplo, Se voce clicar as setas do seu teclado, avera um alerta mostrando qual seta voce clicou
